# تشغيل التخزين على Hostinger

المشروع الآن يدعم التخزين في MariaDB/MySQL بدل الاعتماد على بيانات المتصفح فقط.

## 1. إنشاء قاعدة البيانات

من hPanel في Hostinger:

1. افتح Databases.
2. أنشئ MariaDB/MySQL database جديدة.
3. افتح phpMyAdmin.
4. اعمل Import للملف:

```text
api/schema.sql
```

## 2. ضبط الاتصال

افتح:

```text
api/config.php
```

وغيّر القيم:

```php
const DB_HOST = 'localhost';
const DB_NAME = 'اسم قاعدة البيانات';
const DB_USER = 'اسم مستخدم القاعدة';
const DB_PASS = 'كلمة مرور القاعدة';
```

## 3. رفع الملفات

ارفع كل ملفات المشروع إلى فولدر الـ subdomain، غالبًا داخل:

```text
public_html/subdomain-folder
```

وتأكد أن هذه المجلدات موجودة:

```text
api
assets
uploads
```

## 4. ما الذي يتم تخزينه؟

يتم مزامنة بيانات المنصة العامة تلقائيًا مع قاعدة البيانات:

- الطلاب
- الكورسات
- الدروس والمقاطع
- الاختبارات
- محاولات الاختبارات
- المدفوعات
- أكواد التفعيل
- إعدادات المنصة

الحساب الحالي للطالب لا يتم رفعه كبيان عام، حتى لا ينتقل تسجيل دخول طالب إلى طالب آخر.

## 5. رفع PDF والصور

عند رفع PDF من لوحة التحكم، سيحاول الموقع رفع الملف إلى:

```text
api/upload.php
```

ثم يخزن الرابط داخل بيانات الدرس. الأنواع المسموحة حاليًا:

```text
pdf, jpg, jpeg, png, webp, gif
```
