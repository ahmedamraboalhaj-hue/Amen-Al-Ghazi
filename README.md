# منصة الأمين التعليمية 🎓

منصة تعليمية احترافية لتعلم اللغة العربية مع واجهة مستخدم عصرية وميزات متقدمة.

## الصفحات المتاحة 📄

### 1. **الصفحة الرئيسية** `index.html`
- عرض منجذب للمنصة مع قسم البطل (Hero)
- شريط ملاحة ثابت مع روابط الصفحات
- قسم الإحصائيات
- عرض الكورسات المتاحة
- قسم المميزات الرئيسية
- آراء الطلاب (Testimonials)
- تذييل الصفحة مع معلومات التواصل
- زر WhatsApp عائم
- دعم المواضيع الفاتحة والغامقة

### 2. **نموذج التسجيل** `register.html`
- تسجيل جديد بهاتف وكلمة مرور
- التحقق من صحة البيانات
- حفظ البيانات في localStorage
- التحقق من عدم تكرار رقم الهاتف
- واجهة جميلة وسهلة الاستخدام

### 3. **نموذج الدخول** `login.html`
- تسجيل دخول بهاتف وكلمة مرور
- حساب إدمن خاص: `01000000000` / `admin123`
- التحقق من بيانات الدخول
- إعادة توجيه للوحة التحكم

### 4. **لوحة التحكم** `dashboard.html`
- **للمسؤولين (Admin)**:
  - 📊 نظرة عامة مع إحصائيات ملخصة
  - 👨‍🎓 إدارة الطلاب (CRUD كامل)
  - 📚 إدارة الكورسات
  - 📈 التحليلات والرسوم البيانية
  - 💳 إدارة المدفوعات
  - 💬 الرسائل والإشعارات
  - ⚙️ إعدادات المنصة
  
- **للطلاب (Student)**:
  - 📚 عرض الكورسات المتاحة
  - 📝 تسجيل في الكورسات
  - 📊 تتبع التقدم الدراسي

### 5. **الاختبارات** `tests.html`
- عرض 8 اختبارات متنوعة
- بطاقات اختبار مع معلومات التفاصيل
- حالات مختلفة: مكتمل، قيد الإجراء، لم يبدأ
- نافذة منبثقة لإجابة الأسئلة
- نظام المؤقت (Timer)
- عرض النتائج والدرجات
- إحصائيات شاملة للأداء

### 6. **الدروس** `lessons.html` (الجديد)
- عرض محتوى الدرس الفردي
- مشغل فيديو (placeholder)
- قسم الموارد والملفات للتحميل
- نظام ملاحظات شخصية مع localStorage
- شريط جانبي بقائمة الدروس
- عداد التقدم
- قسم النصائح والإرشادات
- اختبار تدريبي

### 7. **الشهادات والإنجازات** `certificates.html` (الجديد)
- **علامة تبويب الشهادات**: عرض الشهادات الممنوحة
  - 3 شهادات نموذجية
  - تحميل وتشارك الشهادات
  - عرض نسب النجاح

- **علامة تبويب الإنجازات**: نظام الإنجازات
  - 12 إنجاز (مفتوحة ومقفلة)
  - رموز توضيحية
  - متطلبات كل إنجاز

- **علامة تبويب التقدم**: تتبع التقدم
  - أشرطة تقدم لكل كورس
  - نسب الإنجاز
  - رسم بياني للتقدم الأسبوعي

### 8. **استكشاف الدورات** `explore.html` (الجديد)
- شبكة عرض الدورات (150+ دورة متاحة)
- فلاتر حسب المستوى: مبتدئ، متوسط، متقدم
- بطاقات كورس مع:
  - أيقونة تعليمية
  - عدد الطلاب
  - التقييمات ⭐
  
- قسم الميزات: 6 ميزات رئيسية
- إحصائيات عامة عن المنصة
- دعوة للتسجيل (CTA)

### 9. **الملف الشخصي** `profile.html` (الجديد)
- **علامة تبويب بيانات الملف**:
  - الاسم الأول والأخير
  - رقم الهاتف
  - الصف الدراسي
  - النبذة الشخصية

- **علامة تبويب الإحصائيات**:
  - ساعات التعلم
  - دورات مكتملة
  - دروس مكتملة
  - متوسط النتيجة
  - النقاط المكتسبة
  - الشهادات

- **علامة تبويب النشاط**:
  - آخر الأنشطة والإنجازات
  - الدورات المكتملة
  - التقدم الأسبوعي

- **علامة تبويب الإعدادات**:
  - تنبيهات الإشعارات
  - إعدادات الخصوصية
  - أمان الحساب

## بيانات الدخول 🔑

### حساب المسؤول (Admin):
- **الهاتف**: `01000000000`
- **كلمة المرور**: `admin123`

### الطالب النموذجي:
- يمكنك التسجيل كطالب جديد عبر صفحة التسجيل

## هيكل البيانات 📦

البيانات تُخزن في `localStorage` بالمفاتيح التالية:

- `alamin_users`: قائمة المستخدمين
- `alamin_courses`: قائمة الكورسات
- `alamin_current`: المستخدم الحالي
- `alamin_theme`: المظهر المختار
- `alamin_settings`: إعدادات المنصة
- `alamin_pending_payments`: المدفوعات المعلقة
- `alamin_lesson_notes`: ملاحظات الدروس
